Gleice Discord Bot

Projeto pessoal criado para explorar e expandir meus conhecimentos em programação. Através dele, estou aprendendo a desenvolver e integrar diversas tecnologias.

Adicionar
!comandos
- Apresenta uma lista com os comandos disponíveis do bot.
!sort member
- Realiza um sorteio entre os nomes fornecidos, separados por espaço.
!dado X
- Role um dado de X faces (onde X é o número que você escolher).
!purge X
- Exclui as X mensagens mais recentes do canal. Utilize com cautela!
!falar texto
- Faça o bot repetir exatamente o que você digitar.
!somar x y
- Faça o bot somar dois números e te mostrar o resultado.
